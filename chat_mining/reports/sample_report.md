- `未在第一时间澄清是哪个区域`: 1
- `重置无效后没二次诊断（如查后端 log/状态），直接归为'平台负载'`: 1
- `首响应'容器默认没网'给得有点武断（未先看具体报错域名）`: 1
- `未给出查 AIHub 镜像的具体路径，导致用户'没找到'后多轮反复`: 1
- `首响应'啥意思'语气过于随意`: 1

### 决策分支汇总

- GPU余量充足但创建失败 → 后台查 CPU/内存余量 → 资源池整体紧张则建议等待或换区
- OpenClaw 环境异常 → 是否能进控制台 → 休眠→重置按钮
- Module 冲突 → module purge → 重新 source/load
- 外网访问受限 → 区域是否开国际带宽 → 否：给镜像/代理 + 升级业务
- OpenClaw 401 → 登账号试 → 重置 → 仍异常：查后端服务状态/负载（不要止步'多试几次'）
- 客户端连不上服务 → 服务端自测 → 通过则换设备/网络隔离客户端因素
- 容器外网失败 → 是否申请过代理 → 否:申请；是:source 激活 → 仍失败:登账号 + 看缺库
- 外网下载慢 → 是否有平台镜像 → 是:走 AI 社区镜像；否:换时段或平台带宽申请
- 活动赠送类咨询 → 技术 or 运营 → 跨界则 handoff，不要承诺到账时间

### 触发短语索引（症状→用户口语）

- **资源/容量类**: 有卡但创建失败 / 显示没卡 / 明明有卡开机失败
- **报错/异常类**: 422 Model Not Exist / 更换大模型失败 / 龙虾用不了
- **环境配置类**: Module conflicts / source 环境出错 / module load 报错
- **连接/网络类**: 连不上 huggingface / github 慢/连不上 / 外网访问 / 连不上集群 / gateway 连不上 / 之前可以现在不行 / ProxyError / Max retries exceeded
- **操作不会用类→报错/异常类**: 401 / 云端龙虾报错 / Model Not Exist
- **性能/卡顿类**: 下载慢 / 网速慢 / ollama 拉取慢
- **权限/账号类→产品咨询类**: Tokens 没到账 / 活动赠送 / 薅羊毛
